HTML

  • The head element contains information about the webpage.
  • The body element represents the visible content shown to the user.
  • You can shortcut HTML DOCTYPE with the command Shift + 1 then Enter.

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.

Git

Git/GitBash Commands

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • git checkout branch-name: switches to that branch-name
  • git add -A: used to commit all files in the working branch
  • git commit -m "short description": used to associate a message with our commit
  • git pull origin main: pulls information from the main branch to make the current/working branch up to date
  • git push origin branch-name: pushes information from the working branch into the main branch

Git Advisory Notes

  • It is a good habit to always git pull origin main the base branch after you create a new feature branch and before pushing/commiting up any code

When opening a new branch from the main branch:

  1. git pull origin main
  2. code .
  3. change whatever you need to change and then save
  4. git status
  5. git add -A
  6. git commit -m "(message)"
  7. git pull origin main
  8. git push origin branch-name
  9. open the gitHub repo and merge
  10. go back to your terminal and git checkout main
  11. update main branch with the command git pull origin main

JavaScript

  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • Ctrl + F2 = Change all occurances but be careful when using this shortcut as it may select unwanted selections
